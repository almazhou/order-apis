post /products

As a admin, I'd like to add one product to this online shop so that I can sell this product to customer.

  • Request
  • Response

Body

Type: application/json

Schema:

{
  "$schema": "http://json-schema.org/schema",
  "type": "object",
  "properties": {
    "name": {
      "type": "string"
    },
    "category":{"type":"string"},
    "description": {
      "type": "string"
    },
    "imageUrl": {
      "type": "string"
    },
    "sizes":{
      "type":"array"
    },
    "colors":{
      "type":"array"
    },
    "price":{
      "type":"object",
      "properties":{
        "amount":{
          "type":"double"
        },
        "currency":{
          "type":"string"
        }
      }
    }
  },
  "required": [
    "name",
    "description",
    "category",
    "price"
  ]
}

Example:

{
  "name": "LV bags",
  "category": "Bags",
  "imageUrl": "/image/lv.png",
  "description": "LV bags, luxury, life style",
  "sizes": ["XS","S","M","L","XL"],
  "colors": ["RED","YELLOW","GREEN","PURPLE"],
  "price": {
    "amount": 33.0,
    "currency": "USD"
  }
}

  • Response

/products/{product_id}/current_price get

get

As a user, I'd like to know the current price of this product so that I can decide if it is too expensive.

get /products/{product_id}/current_price

As a user, I'd like to know the current price of this product so that I can decide if it is too expensive.

  • Request
  • Response

URI Parameters

  • product_id: required (string)

HTTP status code 200

Body

Type: application/json

Example:

{
	"id":1,
	"uri":"/products/1/current_price",
	"price":{
		"amount":340,
		"currency":"USD"
	},
	"time":"2014-5-12 12:30:59"
}
